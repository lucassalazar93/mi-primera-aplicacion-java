Este proyecto es una aplicación básica desarrollada en Java utilizando el framework Spring Boot. Su propósito es gestionar usuarios mediante una API REST conectada a una base de datos MySQL. La aplicación implementa operaciones CRUD (Crear, Leer, Actualizar y Eliminar) con validación de datos y persistencia usando Hibernate y JPA.
Funcionalidades principales:
Crear, consultar, y eliminar usuarios.
Validación de datos con Jakarta Validation.
Configuración de conexión a base de datos.


Propósito del proyecto:
Aprendizaje de Spring Boot y desarrollo de APIs RESTful con Java.
Ejemplo práctico para iniciarse en proyectos web backend.
